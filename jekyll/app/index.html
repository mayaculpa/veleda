cool technologies we employ

Grafana

Make use of the leading data visualization tool, with templates that automatically configure themselves depending on the connected sensors.

HAPI

Built on the Hydroponic Automation Platform Intiative that creates distributed computing mesh. This ensures that if any node fails or new ones are added, the system automatically reconfigures itself.
